$('#example4').pager('p', {
  navId: 'nav4',
  navAttach: 'after',
  prevText: 'Previous',
  nextText: 'Next',
  linkText: ['Page 1', 'Page 2', 'Page 3', 'Page 4'],
  linkWrap: '<div></div>'
});

Options

navId
For use in more than one pagers per page, takes a string. Default 'nav'
navClass
Gives the nav of the pagers a CSS class. Default 'nav'
navAttach
Position of the nav. Takes either 'prepend', 'append', 'before' or 'after'. Default 'append'.
highlightClass
CSS class of the highlighted link. Default 'highlight'.
prevText
Text for previous link. Default '&laquo;' which gives '«'
nextText
Text for next link. Default '&raquo;' which gives '»'
linkText
Array of text for links. e.g. ['page 1', 'page 2', ...]
linkWrap
Wraps link in HTML using jQuery wrap method. e.g. '<div></div>' would wrap links in a div layer
height
Sets a min-height (height in IE) for your content. Default works out largest content automagically
